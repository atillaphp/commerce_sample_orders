# Commerce Sample Orders

Development helper module to generate sample orders for testing and development purposes. Quickly seed realistic order data into a Drupal Commerce environment.

Part of the **Soğan Commerce** ecosystem.

## Requirements

- Drupal ^11
- drupal/commerce ^3.3

## Installation

```bash
composer require soganbilisim/commerce_sample_orders:^1.0
```

Enable the module (development environments only):

```bash
drush en commerce_sample_orders
```
